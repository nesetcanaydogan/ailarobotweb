# AILA - Project Context

## Vision
AILA is a customizable, physical, AI-powered agentic RAG assistant robot. It transforms physical retail and service environments by providing a natural language interface that perceives surroundings and guides users toward business objectives.

## Core Identity
- **Design System:** 'Internal Dashboard' aesthetic. Dark mode (Midnight Abyss), frosted glass, Neon Violet accents.
- **Tone:** Professional, high-tech, functional, yet approachable.
- **Key Visual:** A 3D model of AILA that follows the user throughout the page journey via scroll synchronization.

## Tech Stack
- **Frontend:** React 19 (Beta/RC features if applicable), Tailwind CSS v4.
- **3D Engine:** React Three Fiber + Three.js.
- **Backend:** Node.js (Express) providing a secure proxy for GPT-4o and handling lead generation.
- **Database:** PostgreSQL with Prisma for robust data persistence (contacts, demo logs).

## Target Audience
- Retailers (Restaurants, Hospitals, Event Venues).
- Search & Rescue / Defense operations (Modular application).
- Businesses seeking high-engagement physical-digital interaction.
